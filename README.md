# Clipit - Video Editing Platform

A platform to connect video creators with professional editors.

## Features

- Browse verified video editors
- Search for editors by name or specialty
- User authentication (login/signup)
- Responsive design with YouTube-inspired theme

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: SQLite
- Authentication: JWT, bcrypt

## Installation

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the server
4. Open `http://localhost:3000` in your browser

## API Endpoints

- `GET /api/creators` - Get all creators
- `POST /api/signup` - User registration
- `POST /api/login` - User login

## Database

The app uses SQLite database (`clipit.db`) with tables for users and creators.
